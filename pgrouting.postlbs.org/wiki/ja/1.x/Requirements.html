ja/1.x/Requirements

必要条件

pgRoutingのバージョンが1.0RC 以上
(以前のバージョンは、TSP とDriving Distance機能無しではコンパイルできません。)

pgRoutingコア - 最短経路アルゴリズム

  • CMakeクロス･プラットフォーム、オープンソースmakeシステム : バージョン 2.3 以上
  • C 及び C++ コンパイラ
  • PostgreSQL (開発者パッケージを含む) : バージョン 8.0 以上
  • PostGIS : バージョン 1.0 以上
  • The Boost Graph Library (BGL) : astar.hpp を含むバージョン 1.33 以上
     http://www.boost.org/libs/graph/doc/index.html 参照

pgRouting 拡張 - 巡回経路検索アルゴリズム(TSP)を含む

  • The Genetic Algorithm Utility Library (GAUL).
     http://gaul.sourceforge.net 参照

pgRouting 拡張 - 到達圏検索アルゴリズム(Driving Distance)を含む

  • Computational Geometry Algorithms Library (CGAL) : バージョン 3.2 以上
     http://www.cgal.org 参照
